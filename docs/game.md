## 시간 제한

### 프리셋 (일반전 / 랭크전 / 친선전)

| 카테고리 | 프리셋 |
|---------|--------|
| Bullet | 1+0, 1+1, 2+1 |
| Blitz | 3+0, 3+2, 5+0, 5+3 |
| Rapid | 10+0, 10+5, 15+10 |

> 카테고리 분류 기준: 총 시간 = 기본 시간 + (40 × 인크리먼트)
> - Bullet: < 3분
> - Blitz: 3~10분 미만
> - Rapid: 10~30분

### 커스텀 (친선전에서만)

- 기본 시간 + 인크리먼트 자유 설정

### Classical

- 현재 미지원
- 추후 대회 모드 또는 특정 Elo 이상에서 개방 검토

## 매칭 방식

### 친선전

1. 방 생성 (시간 제한 설정)
2. 초대 링크 생성
3. 상대가 링크로 참가
4. 게임 시작

### 일반전 / 랭크전

1. 시간 제한 프리셋 선택
2. 매칭 큐 진입 (시간 제한별 큐 분리)
3. 상대 매칭 시 게임 시작

## 레이팅

- Elo 기반
- 랭크전에서만 변동
- 경기 결과는 추후 블록체인에 기록하여 무결성 보장

## 체스 로직

- chess.js 사용 (직접 구현 금지)
- 실시간 통신: Socket.IO
